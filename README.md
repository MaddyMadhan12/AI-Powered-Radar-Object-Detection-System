# AI-Powered-Radar-Object-Detection-System
AI-Powered Radar Object Detection &amp; Visualization System detects objects using an ultrasonic sensor on a rotating servo. Arduino sends distance and angle data to a Python app that displays a live radar screen. The project demonstrates real-time sensing, serial communication, and visualization inspired by robotics and aerospace radar.

# AI Powered Radar Object Detection System

An intelligent radar system that detects and visualizes objects in real time using Arduino, Ultrasonic Sensor, Servo Motor and Python visualization.

## 🚀 Features
- 180° radar scanning using servo motor
- Real-time object detection with HC-SR04 ultrasonic sensor
- Visual radar interface on computer
- Buzzer and LED alert when object detected
- Live distance monitoring

## 🧠 Technologies Used
- Arduino Uno R4 WiFi
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Python (PySerial, OpenCV, Pygame)
- Embedded Systems + AI visualization

## ⚙️ How it Works
1. Servo motor rotates ultrasonic sensor from 0° to 180°
2. Distance data sent from Arduino to computer
3. Python program converts data into radar screen
4. Alerts triggered when object enters danger zone

## 📂 Project Structure
- Arduino Code → Sensor + Servo control
- Python Code → Radar Visualization
- Hardware → Real-world object detection

## 🎯 Applications
- Security systems
- Robotics obstacle detection
- Smart surveillance
- Autonomous vehicles concept
